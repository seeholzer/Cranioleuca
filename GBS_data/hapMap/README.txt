HapMap.fas.txt
This file contains the fasta format sequences for all tag pairs, the name of the sequence indicates the Tag Pair membership and the tag length. Tags shorter than 64 bp have been padded with tailing A.

 HapMap.hmc.txt
This file contains the genotype information for each individual for each tag pair, along with the total allele counts and the allele frequency

HapMap.hmp.txt
This file is a hapmap file that can be imported into TASSEL,or any other program that handle hapmap format.
The allele order in this file is major/minor *not* reference/alternative
The hapmap file HapMap.filtered.hmp.txt contains filtered SNP calls (see report for filtering parameters
